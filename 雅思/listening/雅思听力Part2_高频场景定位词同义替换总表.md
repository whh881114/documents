# 雅思听力 Part 2 高频场景、定位词、同义替换总表

## 使用说明

这份只保留 Part 2 内容，方便单独看、单独背。

Part 2 的核心，不是单纯“听到熟词就选”，而是：

**先判断场景 → 再抓路线、设施、安排和规则 → 再等同义替换 → 再防方向干扰和提前抢选。**

Part 2 常见题型包括：
- 地图题
- 选择题
- 表格 / 笔记补全
- 活动 / 场馆 / 服务说明

## 二、Part 2 高频场景、定位词、同义替换

### 1. 旅游景点 / 博物馆 / 公园 / 展馆介绍

#### 常见内容
- 某个景点介绍
- 游客中心说明
- 展馆开放信息
- 园区设施
- 参观路线
- 注意事项

#### 高频定位词
- entrance
- exit
- ticket office
- information desk
- guided tour
- opening hours
- café
- gift shop
- toilets
- car park
- exhibition
- path
- bridge
- hall
- gallery

#### 高频同义替换
- **entrance** = main gate / way in
- **exit** = way out
- **information desk** = reception / help desk
- **guided tour** = tour with a guide
- **opening hours** = when it opens / visiting times
- **gift shop** = souvenir shop
- **toilets** = restrooms / washrooms
- **car park** = parking area
- **exhibition** = display / show
- **path** = track / walkway
- **bridge** = crossing
- **hall** = main room / area

#### 做题提醒
这类题很常考：
- 设施在哪
- 哪个区域干什么
- 哪条路线怎么走
- 哪个时间开放
- 哪个地方有限制

### 2. 地图题 / 平面图题 / 路线题

#### 常见内容
- 建筑内部布局
- 校园地图
- 公园路线
- 新旧设施位置变化
- 从某入口走到某个点

#### 高频定位词
- on the left
- on the right
- opposite
- next to
- behind
- in front of
- at the end of
- in the corner
- beside
- between
- across from
- north / south / east / west
- entrance
- reception
- stairs
- lift
- corridor

#### 高频同义替换
- **on the left** = to your left / left-hand side
- **on the right** = to your right / right-hand side
- **opposite** = across from / facing
- **next to** = beside / by
- **behind** = at the back of
- **in front of** = before you reach
- **at the end of** = right at the far end
- **in the corner** = at the corner
- **between** = in the middle of
- **stairs** = staircase
- **lift** = elevator
- **corridor** = hallway / passage

#### 做题提醒
地图题最容易死在这几个地方：
1. 一上来没确定起点
2. 左右方向跟丢
3. 只听名词，不听移动逻辑
4. 旧位置和新位置混掉
5. 听到熟词太激动，提前选了

#### 地图题必须盯住的路标词
- first
- then
- after that
- before
- just beyond
- directly opposite
- on either side of
- at the far end
- immediately after

#### 常见内容
- 社区活动
- 周末节日活动
- 夏令营安排
- 市场 / 展会安排
- 志愿者活动说明

#### 高频定位词
- schedule
- programme
- event
- performance
- workshop
- speaker
- start time
- venue
- booking
- registration
- suitable for
- free admission
- lunch break

#### 高频同义替换
- **schedule** = timetable / plan
- **programme** = agenda
- **event** = activity / occasion
- **performance** = show
- **workshop** = training session / practical session
- **speaker** = presenter
- **venue** = location / place
- **booking** = reservation
- **registration** = sign-up
- **suitable for** = intended for / good for
- **free admission** = no entrance fee
- **lunch break** = midday break

#### 做题提醒
别光盯时间。  
很多题其实在考：
- 活动目的
- 适合什么人
- 报名方式
- 哪个环节最值得注意

### 4. 设施介绍 / 服务说明 / 规则通知

#### 常见内容
- 健身中心介绍
- 图书馆服务介绍
- 游客中心规则
- 营地设施说明
- 课程中心使用规范

#### 高频定位词
- available
- facilities
- equipment
- members
- visitors
- rules
- booking required
- opening times
- access
- available for hire
- safety
- prohibited

#### 高频同义替换
- **available** = provided / offered
- **facilities** = services / amenities
- **equipment** = tools / gear
- **members** = registered users
- **visitors** = guests
- **rules** = regulations
- **booking required** = must be reserved in advance
- **access** = entry / use
- **available for hire** = can be rented
- **safety** = security / protection
- **prohibited** = not allowed / forbidden

#### 做题提醒
这种题很爱出：
- 什么可以做
- 什么不可以做
- 什么时候可以用
- 谁可以用
- 需不需要提前预约

## 四、最后的核心提醒

Part 2 最容易出问题的，不是单词本身，而是这几个地方：

1. **起点没抓住，地图方向直接乱掉**
2. **只听名词，不听路线和顺序**
3. **听到熟词太兴奋，提前选了干扰项**
4. **只会等原词，不会等替换说法**
5. **安排、规则、时间这些信息听到了，但没和题目对上**

所以做 Part 2 时，脑子里最好始终跑这条线：

**先看题目判断场景，再盯住起点、方向、安排和规则，然后再去等真正的答案。**
