# 雅思听力 Part 3 高频场景、定位词、同义替换总表

## 使用说明

这份只保留 Part 3 内容，方便单独看、单独背。

Part 3 的核心，不是单纯“听懂几个学术词”，而是：

**先判断讨论主题 → 再盯转折、态度、分工和最终决定 → 再抓定位词和同义替换。**

Part 3 常见题型包括：
- 选择题
- 匹配题
- 表格 / 流程补全
- 学生讨论 / 导师反馈 / 小组分工

---

## 一、Part 3 的核心特点

## Part 3
### 本质
- 两个人或三个人讨论
- 常见是学生和导师、学生之间讨论作业 / 研究 / 课程 / 项目
- 是“半学术”场景

### 你要抓什么
- 讨论主题
- 分工
- 原因
- 观点变化
- 建议
- 问题与解决办法
- 谁负责什么
- 老师真正强调什么

### 常见难点
- 多人说话，容易乱
- 同义替换明显增加
- 态度、观点、转折很多
- 你以为在听细节，其实在考“理解关系”

## 二、Part 3 高频场景、定位词、同义替换

### 1. 课程作业讨论

#### 常见内容
- 选题
- 作业要求
- 资料搜集
- 分工
- 遇到的问题
- 修改建议

#### 高频定位词
- assignment
- topic
- research
- sources
- deadline
- draft
- feedback
- structure
- introduction
- conclusion
- reference
- notes

#### 高频同义替换
- **assignment** = project / piece of work
- **topic** = subject / theme
- **research** = investigation / study
- **sources** = materials / references
- **deadline** = due date
- **draft** = first version
- **feedback** = comments / advice
- **structure** = organisation
- **introduction** = opening section
- **conclusion** = final section
- **reference** = source citation
- **notes** = points / written ideas

#### 做题提醒
Part 3 不是“听到 topic 就写 topic”。  
它常考的是：
- 为什么选这个题
- 老师觉得哪里不合适
- 还缺什么
- 谁去做什么

---

### 2. 小组项目 / 分工讨论

#### 常见内容
- 两三个人讨论 group project
- 分工安排
- 哪部分谁做
- 时间表怎么排
- 遇到的问题和解决方案

#### 高频定位词
- group project
- role
- responsibility
- task
- data
- presentation
- survey
- interview
- prepare slides
- analyse
- collect information
- divide the work

#### 高频同义替换
- **role** = part
- **responsibility** = duty
- **task** = job
- **presentation** = talk / oral report
- **survey** = questionnaire
- **interview** = talk to participants
- **prepare slides** = make the PowerPoint
- **analyse** = look at / interpret
- **collect information** = gather data
- **divide the work** = share out the tasks

#### 做题提醒
多人对话最容易死在：
- 你只跟住一个人
- 另一个人突然给出关键答案你没听到
- 你没分清“建议”和“最终决定”

---

### 3. 导师 / 老师 / 教授反馈

#### 常见内容
- 老师点评学生草稿
- 研究方向调整
- 让学生补材料
- 提醒方法不对
- 纠正思路

#### 高频定位词
- tutor
- supervisor
- feedback
- improve
- focus on
- too broad
- too narrow
- evidence
- examples
- methodology
- argument
- clearer

#### 高频同义替换
- **improve** = make better / strengthen
- **focus on** = concentrate on
- **too broad** = not specific enough
- **too narrow** = too limited
- **evidence** = support / proof
- **examples** = illustrations
- **methodology** = method / approach
- **argument** = main point / line of reasoning
- **clearer** = easier to follow / more precise

#### 做题提醒
老师的话通常更值钱。  
很多答案不是学生说出来的，而是老师最后定调：
- 这题太大
- 这个方法更合适
- 这个部分需要删
- 你需要更多例子

---

### 4. 课程选择 / 学习计划 / 研究计划

#### 常见内容
- 选课
- 选研究主题
- 做学期计划
- 讨论实验或调查怎么做

#### 高频定位词
- optional course
- compulsory
- module
- seminar
- lecture
- lab work
- field trip
- timetable
- research question
- participants
- sample size
- interview questions

#### 高频同义替换
- **optional** = elective
- **compulsory** = required / mandatory
- **module** = course unit
- **seminar** = discussion class
- **lecture** = formal class talk
- **lab work** = practical work
- **field trip** = visit for study
- **participants** = people taking part
- **sample size** = number of people studied

#### 做题提醒
这类题常考：
- 为什么选这个课
- 哪个模块更有帮助
- 研究对象是谁
- 方法为什么改了

## 三、Part 3 高频逻辑词与态度词

### 1. 转折信号词
- but
- however
- actually
- in fact
- although
- on the other hand
- the problem is
- what worries me is

#### 提醒
转折后面，常常才是答案。

---

### 2. 建议信号词
- maybe we should
- it might be better to
- why don't we
- perhaps
- we could
- I think the best option is

#### 提醒
这些通常引出“方案比较”。

---

### 3. 同意 / 不同意
- that's true
- I agree
- not really
- I'm not sure
- that won't work
- good point
- exactly

#### 提醒
题目常考最终决定，不是第一个想法。

---

### 4. 态度与评价
- useful
- practical
- unrealistic
- effective
- time-consuming
- confusing
- appropriate
- convincing

#### 提醒
Part 3 常常不是考事实，而是考“他们怎么看”。

## 四、Part 3 题型专属提醒

### 1. 匹配题
#### 常见内容
- 谁负责什么
- 谁提了什么观点
- 哪个资料对应哪个问题
- 哪种方法对应哪项用途

#### 你要重点听
- 人名
- 代词
- 评价词
- 最终决定词

#### 常见坑
- 同一个人提了多个想法
- 先否定一个，再换一个
- 你没跟住代词 he / she / they

---

### 2. 选择题
#### 你要重点听
- 原因
- 目的
- 老师真正建议什么
- 他们最后决定了什么

---

### 3. 表格 / 流程补全
#### 你要重点听
- 步骤顺序
- 分类关系
- 因果关系
- 谁负责哪一步

---

---

## 五、最后的核心提醒

Part 3 最容易出问题的，不是词难，而是这几个地方：

1. **多人说话时，只跟住了一个人**
2. **没跟住转折，结果把前面的想法当成答案**
3. **没分清“建议”“讨论”和“最终决定”**
4. **代词 he / she / they 一出来，人物关系就乱了**
5. **只听单词，不听态度，不听逻辑关系**

所以做 Part 3 时，脑子里最好始终跑这条线：

**先判断谁在讨论什么，再盯转折和态度，最后再确认他们真正决定了什么。**
