# 巫师3：行于任务，思于故事



## 前言
巫师3有很多支线任务，在这些任务中，有那么几个让我印象深刻。


## 支线任务：天长地久

### 背景
夜晚时分，博克莱尔的墓园里出现了令人不安的噪音。人们原本以为是盗墓贼或死灵怪物在作祟，于是请猎魔人前往调查。
杰洛特到场后发现，问题并不在于怪物——而是一对死后仍争吵不休的亡灵夫妇 Louis de Corentin 和 Margot de Corentin。
他们尖锐的争吵在夜晚不断回荡，扰得整座墓园不得安宁。

### 台词
```
Margot：你寧願整天閒閒沒事做、把時間都拿去玩紙牌，簡直跟...跟個矮人一樣！噁心！呸！
 Louis：沒錯！總比站在雷比歐達神廟的門口，從早到晚唸禱文好多了！
Margot：忘恩負義的傢伙！我是在為你祈禱！希望你能戒掉賭博的壞毛病！
 Louis：那你還真成功！看來天神聽到你的願望了！那你呢！你又在這裡幹嘛？這是我們的家務事！
 傑洛特：你們究竟在吵什麼？
 Louis：滾開！這不甘你的事！
 傑洛特：我可以幫你們解決。
Margot：我不知道你是誰，但拜託你幫幫我！我不想永遠被困在這野蠻人旁邊！請立刻把他趕出我的墓室！
 傑洛特：別激動。這是怎麼回事？
 Louis：這是我的墓室！這潑婦連死了都不肯放過我！永無止境的嘮叨，沒有比這更慘的事了！
Margot：我根本是眼睛瞎了才會嫁給他！他根本是昆特牌的奴隸，連小孩都願意拿去賭！
 Louis：就那麼一次而已，可惡！而且那只是開玩笑！你要我解釋幾次？那又是誰把我們的錢拿去做什麼愚蠢的雷比歐達雕像？
Margot：我是出自虔誠之心！我以為先知會淨化這片瀆神的賭博之地！我怎麼知道那完全是騙局？
 Louis：我不是警告過你嗎？不下五百次啊？
 傑洛特：安靜！閉嘴！兩個都閉嘴！
 傑洛特：就名義上來說，這墓室是誰的？
 Louis：還要問嗎？是我的，是科倫汀家族的。你不識字嗎？
Margot：沒錯，但他是用我的嫁妝把這座墓室從債主手中買回來的。
 傑洛特：聽著，如果這裡要恢復寧靜，你們其中一個就得走。得把你們其中之一帶走才行。
Margot：沒錯，把他趕走！拜託！趕他走，我就...我就告訴你我把他的幸運昆特牌藏在哪裡。
 Louis：什麼？我就知道是你！現在你還要把我趕出自己的墓穴，還用我的牌當報酬？
 Louis：該死！要是我拿那些牌去參加貝哈文的錦標賽，我就不會死了！都是你的錯，你這個黃臉婆，我被他們捅死全都是你的錯！
Margot：他們是因為你欠債才殺了你，跟我根本無關！
 Louis：別理這個滿腦子聖書的瘋婆娘了吧？我知道她把那些牌藏在哪裡。只要你先把她趕走，我就告訴你！
 傑洛特：我要考慮一下。
```

### 选择与结局
- 选择：将Margot移出墓园。

- 理由：因为我在游戏中经常玩昆特牌，这已经是一件稀松平常的事，并且已经完成了昆特牌相关的成就（卡牌收藏家，昆特成瘾，尽遣王牌，昆特牌大师），
所以我很能体会Louis的愤怒。你把他的好牌藏起来了，那他还怎么赢，还因此丢掉了性命，你真是一个坏婆娘。另外，在台词中，Louis也 声明了，
他并没有拿自己孩子当赌注，只是开玩笑，也没到丧心病狂的地步，而我也正好想知道Louis有哪些好牌，符合我在游戏中的利益。至于Margot方面，
她私自将夫妻的共同财产参与雷比欧达的雕像建造，结果还被骗了，Louis在此之前劝说过很多次了，结果就是不听，也能感受到Louis的无奈。

- 结局：回来不见Louis，只能逼他现身，而他根本不知道昆特牌藏在哪里，无法兑现之前的承诺，我非常气愤，竟然敢骗猎魔人，真是吃了熊心豹子胆，
我都想让他见见"布拉维坎的屠夫"的外号可不是白叫的。替代的奖励是一把剑，这个奖励对我来说已经无关紧要了。

### 思考
- 我和Louis其实都是赌徒，赌徒之间很能共情。假如在赌博中赢了钱，从来不承认这是运气，而是，我就是天选之子的感觉；如果输钱了，就怪身边的
一切，哪怕是一只苍蝇在旁边飞过，那都要问候它八辈子祖宗。总之，这一切都不是我的错。

- 赌徒什么时候才离开牌桌？那一定是倾家荡产的时候，《活着》电影中的富贵就给出了答案。再反观Louis说的玩笑话——把孩子当赌注，其实这已经到
了危险边缘，他只是现在还没有把孩子当赌注而已。
