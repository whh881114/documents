# 巫师3：行于任务，思于故事



## 前言
巫师3有很多支线任务，在这些任务中，有那么几个让我印象深刻。


## 天长地久
```
Margot：你寧願整天閒閒沒事做、把時間都拿去玩紙牌，簡直跟...跟個矮人一樣！噁心！呸！
 Louis：沒錯！總比站在雷比歐達神廟的門口，從早到晚唸禱文好多了！
Margot：忘恩負義的傢伙！我是在為你祈禱！希望你能戒掉賭博的壞毛病！
 Louis：那你還真成功！看來天神聽到你的願望了！那你呢！你又在這裡幹嘛？這是我們的家務事！
 傑洛特：你們究竟在吵什麼？
 Louis：滾開！這不甘你的事！
 傑洛特：我可以幫你們解決。
Margot：我不知道你是誰，但拜託你幫幫我！我不想永遠被困在這野蠻人旁邊！請立刻把他趕出我的墓室！
 傑洛特：別激動。這是怎麼回事？
 Louis：這是我的墓室！這潑婦連死了都不肯放過我！永無止境的嘮叨，沒有比這更慘的事了！
Margot：我根本是眼睛瞎了才會嫁給他！他根本是昆特牌的奴隸，連小孩都願意拿去賭！
 Louis：就那麼一次而已，可惡！而且那只是開玩笑！你要我解釋幾次？那又是誰把我們的錢拿去做什麼愚蠢的雷比歐達雕像？
Margot：我是出自虔誠之心！我以為先知會淨化這片瀆神的賭博之地！我怎麼知道那完全是騙局？
 Louis：我不是警告過你嗎？不下五百次啊？
 傑洛特：安靜！閉嘴！兩個都閉嘴！
 傑洛特：就名義上來說，這墓室是誰的？
 Louis：還要問嗎？是我的，是科倫汀家族的。你不識字嗎？
Margot：沒錯，但他是用我的嫁妝把這座墓室從債主手中買回來的。
 傑洛特：聽著，如果這裡要恢復寧靜，你們其中一個就得走。得把你們其中一人帶走才行。
Margot：沒錯，把他趕走！拜託！趕他走，我就...我就告訴你我把他的幸運昆特牌藏在哪裡。
 Louis：什麼？我就知道是你！現在你還要把我趕出自己的墓穴，還用我的牌當報酬？
 Louis：該死！要是我拿那些牌去參加貝哈文的錦標賽，我就不會死了！都是你的錯，你這個黃臉婆，我被他們捅死全都是你的錯！
Margot：他們是因為你欠債才殺了你，跟我根本無關！
 Louis：別理這個滿腦子聖書的瘋婆娘了吧？我知道她把那些牌藏在哪裡。只要你先把她趕走，我就告訴你！
 傑洛特：我要考慮一下。

```
