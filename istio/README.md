# istio

## 部署模式
```
                           +-----------------------------+
                           |       master 集群           |   <--- 单一控制平面
                           |   安装 Istio control plane  |
                           |     istiod / ingress GW     |
                           +-------------+---------------+
                                         |
              +--------------------------+--------------------------+
              |                          |                          |
              |                          |                          |
   +----------v----------+    +----------v----------+    +----------v----------+
   |     bj 集群         |    |     sh 集群         |    |     gz 集群         |
   |   Remote Cluster    |    |   Remote Cluster    |    |   Remote Cluster    |
   | 没有安装 istiod     |     | 没有安装 istiod     |    | 没有安装 istiod     |
   | 有 east-west GW     |    | 有 east-west GW     |    | 有 east-west GW     |
   +----------+----------+    +----------+----------+    +----------+----------+
              |                          |                          |
              +-----------+ 通过 ServiceEntry / VirtualService 定义 +----------+
                          |        实现跨集群服务调用                |
                          +----------------------------------------+


```
