# 巫师3：行于任务，思于故事



## 前言
巫师3有很多支线任务，在这些任务中，有那么几个让我印象深刻。


## 天长地久
```
Margot：你宁愿整天闲闲没事做，把时间都拿去玩纸牌，简直跟...跟个矮人一样！恶心！呸！
Louis：没错！总比站在雷比欧达神庙的门口，从早到晚念祷文好多了！
Margot：忘恩负义的家伙！我是在为你祈祷！希望你能戒掉赌博的坏毛病！
Louis：那你还真成功！看来天神听到你的愿望了！那你呢！你又在这里干嘛？这是我们的家务事！
杰洛特：你们究竟在吵什么？
Louis：滚开！这不干你的事！
杰洛特：我可以帮你们解决。
Margot：我不知道你是谁，但拜托你帮帮我！我不想永远被困在这野蛮人旁边！拜托你马上把他赶出我的墓室？
杰洛特：别激动。这是怎么回事？
Louis：这是我的墓室！这泼妇连死了都不肯放过我！永无止境的唠叨，没有比这更惨的事了！
Margot：我根本是眼睛瞎了才会嫁给他！他根本是昆特牌的奴隶，连小孩都愿意去赌！
Louis：就那么一次而已，可恶！而且那只是开玩笑！你要我解释几次？那又是谁把我们的钱拿去做什么愚蠢的雷比欧达雕像？
Margot：我是出自虔诚之心！我以为先知会净化这片渎神的赌博之地！我怎么知道那完全是骗局？
Louis：我不是警告过你吗？不下五百次啊？
杰洛特：安静！闭嘴！两个都闭嘴！就名义上来说，这墓室是谁的？
Louis：还要问吗？是我的，是科伦汀家族的。你不识字吗？
Margot：没错，但他是用我的嫁妆把这座墓室从债主手中买回来的。
杰洛特：听着，如果这里要恢复宁静，你们其中一个就得走。得把你们其中一个带走才行。
Margot：没错，把他赶走！拜托！赶他走，我就...我就告诉你我把他的幸运昆特牌藏在哪里。
Louis：什么？我就知道是你！现在你还要把我赶出自己的墓穴，还用我的牌当报酬？
Louis：该死！要是我拿那些牌出战贝哈文的锦标赛，我就不会死了！都是你的错，你这个黄脸婆，我被他们捅死全都是你的错！
Margot：他们是因为你欠债才杀了你，跟我根本无关！
Louis：别理这个满脑子圣书的疯婆娘了吧？我知道她把那些牌藏在哪里。只要你先把她赶走，我就告诉你！
杰洛特：我要考虑一下。
```
