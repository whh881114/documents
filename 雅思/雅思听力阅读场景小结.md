# 雅思听力常见场景总览

雅思听力的场景**高度集中**，更像固定题库反复换皮。可以粗分为两大“宇宙”：

## 1. 生活行政宇宙（多见于 Section 1 / Section 2）
核心主题：**办事、预约、预订、报名、咨询、投诉**（现实生活里也常见的琐碎任务）

### 高频场景
- **租房 / 住宿**：看房、押金、租约、设施、室友规则
- **旅游 / 交通**：订票、时刻表、路线、行李、退款改签
- **办卡 / 付费**：会员、费用、收据/回执、付款方式
- **客服 / 投诉**：订单号、损坏、补偿、确认函
- **健康预约**：症状、预约、处方、复诊（偶尔出现）
- **活动 / 景点**：门票、开放时间、导览、报名

### 特点
- **数字密度爆炸**：时间、日期、价格、电话、地址、邮编、房间号等
- 因此数字听写训练对 Section 1/2 的收益最大

## 2. 校园学术宇宙（多见于 Section 3 / Section 4）
核心主题：**学习、研究、讲座、作业、课程行政**

### 高频场景
- **选课 / 注册 / 学费**：课程、学分、截止日期、退课、证书
- **作业与研究**：论文结构、研究方法、数据、图表、引用、抄袭
- **导师讨论**：课题方向、时间安排、反馈修改
- **讲座 / 报告**：某个学术主题的介绍（信息密集）

### 特点
- 词汇更偏“学术口吻”，但题目逻辑很套路：**主旨、步骤、因果、对比、分类**
- 前 10–20 秒识别场景后，很多信息可以“预判”会出现什么类型（例如截止日期、费用、流程步骤等）

## 结论
雅思听力场景并不乱：
- **Section 1-2：生活办事表格流**
- **Section 3-4：校园讨论/讲座信息流**

抓住场景触发词（如 rent/deposit/lease 或 assignment/deadline/tutorial），后面做题会明显更稳、更快。

# 雅思 A 类阅读（Academic Reading）的“类型”

你问的“类型”通常有两层：**文章类型**与**题型（Question Types）**。两层都给你列清楚，方便你对号入座。

## 1. 文章类型（Passage Styles）
A 类阅读一般是**3 篇长文章**，来源常见于书籍、期刊、报刊杂志等的非虚构文本，主题偏学术但面向大众。

常见写作风格大致分为：
- **事实说明型（Descriptive / Factual）**
  - 介绍某个现象、历史、流程、定义、分类（信息型）
- **分析解释型（Analytical / Explanatory）**
  - 讲原因机制、影响、模型、证据链（逻辑更强）
- **议论论证型（Discursive / Argumentative）**
  - 给观点、对比立场、反驳、权衡利弊（立场与结构更明显）

常见主题领域（了解即可，不用“背知识”）：
- 科普/技术（生物、医学、工程、AI、能源）
- 环境与地理（气候、生态、农业）
- 社会科学（教育、心理、经济、城市）
- 历史与文化（考古、语言、艺术）
- 商业与管理（组织、创新、市场）

## 2. 题型类型（Question Types）
A 类阅读的题型非常固定，常见包括：
- **Matching Headings**（段落标题匹配）
- **True / False / Not Given**（判断题）
- **Yes / No / Not Given**（观点判断）
- **Multiple Choice**（单选/多选）
- **Matching Information**（信息匹配到段落）
- **Matching Features**（特征匹配：人物/理论/时间等）
- **Matching Sentence Endings**（句子结尾匹配）
- **Sentence Completion**（句子填空）
- **Summary / Notes / Table / Flow-chart Completion**（摘要/笔记/表格/流程图填空）
- **Diagram Labelling**（图示标注）
- **Short Answer Questions**（简答）

## 实战提示（超短版）
- “**Heading**”更考归纳与结构；“**Matching Information/Contain**”更偏定位与同义替换。
- A 类阅读本质是：**定位能力 + 同义替换网 + 段落逻辑**，不是背景知识竞赛。
