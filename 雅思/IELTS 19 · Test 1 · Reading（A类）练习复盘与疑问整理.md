## 2. 通用策略：先读文章还是先看题？

结论：**大多数情况下先看题，再带目的读文章（最省脑子、也更贴合雅思机制）。**

### 推荐流程（最通用）
1. **先扫题目（不做题，只看要找什么）**
   - 圈关键词：人名/年份/地名/专有名词/比较词/极限词（only/mostly/never）
   - 识别题型：填空、段落匹配、T/F/NG、YES/NO/NG、多选等
2. **快速扫文章结构（30–90秒）**
   - 只读标题、小标题、每段首句/末句
   - 给每段写 3–7 字“标签”（段落地图）
3. **按题目去定位证据句**
   - 找到证据句就出答案，不把文章当小说精读

### 什么时候需要“先扫文章再做题”？
- **段落标题匹配/Heading matching**：需要先建立段落主旨地图
- **文章极抽象（观点推进强）**：先做 1 分钟结构扫读，避免迷路  
> 但仍不建议“慢读全文再做题”，那是用记忆硬扛检索系统，极耗时。

## 3. 通用题型：TRUE/FALSE/NOT GIVEN 与 YES/NO/NOT GIVEN 区别

### 3.1 TRUE / FALSE / NOT GIVEN（事实判断）
题干在问：**文章里的“事实信息”是否支持这个陈述？**

- **TRUE**：文章明确说了/同义改写，结论一致  
- **FALSE**：文章明确给出相反信息（证据反向打脸）  
- **NOT GIVEN**：文章没有给足信息判断真伪（不能脑补）

关键词：**fact claim（事实断言）**、客观信息、可被证据验证。

### 3.2 YES / NO / NOT GIVEN（观点态度）
题干在问：**作者/说话者是否支持这个观点？**

- **YES**：作者观点支持/赞同题干  
- **NO**：作者观点反对/不赞同题干  
- **NOT GIVEN**：作者没有表态，看不出来立场

关键词：**opinion/attitude（观点/态度）**、立场、评判、主观色彩。

### 3.3 一句话记忆法
- **T/F/NG：事实对不对**
- **Y/N/NG：作者站不站你这边**
- **NG：没证据别脑补；没提 ≠ 反对**

### 3.4 常见翻车点（必须避免）
- 把 **NOT GIVEN 当成 FALSE/NO**  
  - 反对必须有反向证据；没证据就是 NG
- 漏掉限定词：**only / mainly / most / some / in the 1990s**  
  - 限定词一漏，答案可能直接翻转
- YES/NO 题只找“事实”不找“态度信号词”  
  - 支持：argues / suggests / emphasizes / is convinced  
  - 反对：criticizes / doubts / warns against / challenges

## 4. 题型疑问与策略总结（本次遇到的重点）

### 4.1 段落信息匹配题（Which paragraph contains the following information? A–G）

**特征**：题干抽象、同义替换多，容易“看完文章记不住”。

**核心思路**：不是记全文，而是 **段落地图 + 题干信号定位证据句**。

#### 推荐流程（提速版）
1. **先给每段贴标签（30–60秒）**
   - 只读每段：首句/末句 + 扫专有名词、数字、地名
   - 用 3–7 个字写段落主题（如：现代印象/航海条件/国家利用/剿匪行动/安置政策…）
2. **把题干拆成定位信号**
   - 例如：denial / campaign / states use pirates / today view / encouraged not return / close to land
3. **先缩小范围再精读**
   - 先判断“更可能在哪两段”，再回段落里找证据句

> 结论：这题难点不是“英语读不懂”，而是“检索效率”与“同义替换敏感度”。

---

### 4.2 多选题（Choose TWO letters, A–E）

#### 题型特点
- 选项长、细节多、容易被带着全文乱找。
- 正确做法：**不是逐个验证 5 个选项**，而是快速淘汰 3 个。

#### 提速策略
1. **先把选项压缩成 2–3 个关键词标签**
2. **先在文章里锁定范围段落**
   - 题干一般明确范围（例如 Mediterranean inhabitants / ancient Greece / Iliad & Odyssey）
3. **只验证“剩下最可能的两项”**
   - 找到证据句就收工，不要企图把 5 个都证伪/证真

---

### 4.3 Summary Completion（10选6）

**题型体验**：同义替换不好找，做对 3 个。

#### 关键发现
- 很多空可以用 **语法结构 + 固定搭配（collocation）** 先锁定候选，不必死盯同义替换。
- 但最终仍需回原文快速确认，避免“多个选项都语法对”的陷阱。

#### 本次题目（31–36）推理版答案
- 31 = **G** frequent exposure（exposure to misinformation）
- 32 = **J** different ideas（had different ideas about…）
- 33 = **H** mental operation（a distinct mental operation）
- 34 = **B** additional evidence（provide evidence for…）
- 35 = **E** short period（for an extremely short period）
- 36 = **C** different locations（in different locations in the brain）

#### 考试建议（稳妥版两段式）
1. **先用语法/搭配预填**（速度快）
2. **再回原文做点位确认**（每空 10–20 秒）
   - 确认：主旨一致 + 有证据句/同义表达支撑

## 5. 练习方式与载体问题（PDF vs 纸质 / 机考）

- 使用 PDF 来回翻页定位，效率较低，容易耗时间、打断思路。
- 纸质练习的优势：
  - 圈关键词更顺手
  - 段落地图/标记更直观
- 目标：训练到 **60 分钟完成**，而不是 90 分钟慢慢做对。

## 8. 如何精读（把“看懂”变成提分产出）

> 备考阶段的精读不是逐句翻译爽一遍，而是：**用真题做输入积累，同时留下可复用的输出**，让你第二遍、第三遍做题时更快更准。  
> 建议把一套阅读“复用三遍”：第1遍输入精读 → 第2遍方法提速 → 第3遍60分钟实战。

---

### 8.1 精读的目标（验收标准）

精读做完，至少满足这三条（满足才算“有产出”）：

1. 能在 **30秒内定位**错题/慢题的证据句（段落 + 句子范围）
2. 能写出 **≥10–20组**“题干 ↔ 原文”的同义替换（词块/结构都算）
3. 二刷同一篇（只刷错题/慢题）**速度明显提升**，并且 NG/False/Yes/No 判定更果断

---

### 8.2 精读的方式（分层：全文理解 + 关键句产出）

你可以把第1遍当“阅读材料”，允许慢，但要控制节奏：

- **全文理解**：可以逐句读懂（用于长难句训练与降低抽象恐惧）
- **关键句产出**：只对“决定答案”的句子做深加工（这是提分核心）

建议关键句范围：
- 所有错题对应的证据句（必做）
- 所有“做对但耗时很久”的题对应的证据句（很值）
- 每段挑 1 句最抽象/结构最复杂的句子（可选）

---

### 8.3 精读的具体步骤（第1遍：输入精读）

#### Step A：做题（不限时，但要记录用时）
- 先做完整篇题目（可不严格限时）
- 记录：每篇用时、每组题卡在哪（定位/判定/粗心）

#### Step B：逐句读懂（输入）
- 目标：理解每段主旨 + 关键句结构
- 做法：遇到长难句只做三件事（避免写长翻译）
  1) 划主干（S-V-O）
  2) 标从句/插入语/分词结构
  3) 圈出关键逻辑词（however/therefore/while/whereas/in contrast）

> 逐句读懂可以做，但不要变成“全文翻译本”；重点是结构与逻辑。

#### Step C：证据句库（Evidence Bank）——精读的“硬产出”
对每个错题/慢题建立一条记录：

- Q编号 / 题型 / 正确答案
- Evidence（抄1–2句原文）
- Paraphrase（题干关键词 → 原文表达）
- Decision（判定理由一句话：TRUE/FALSE/NG 或 YES/NO/NG）

#### Step D：同义替换地图（Paraphrase Map）——每篇≥15组
从文章与题干里抽取高频替换，写成词块对照（推荐写“短语/结构”，不只写单词）：

- common view → popular image / perception
- campaign to eradicate → crackdown / concerted effort to eliminate
- denial of involvement → rejected accusations / claimed not to be involved
- encouraged not to return → discouraged from going back / deterred from returning
- close to land → coastal waters / near the shoreline
- provide evidence for → support / findings backing

> 目的：训练“见到题干→脑内自动联想到文章可能用什么写法”。

#### Step E：段落地图（Paragraph Map）——30秒版本
- 每段用 3–7 个字写主题标签（A–G）
- 为段落匹配题服务（避免每题从头翻到尾）

#### Step F：题型规则卡（Decision Rules）——只写3–5条
把你这篇最容易犯错的点写成规则（越短越好）：
- NG：无证据不脑补；没提 ≠ 反对
- False/No：必须有反向证据，不是“我觉得不对”
- 段落匹配：先地图再拆题干信号词
- Summary：先语法/搭配锁候选，再回原文点位确认

---

### 8.4 第二遍与第三遍：把精读“变现”为速度

#### 第2遍（方法型，隔24–72小时）
- 目标：把“懂”转成“定位+判定速度”
- 流程：先看题→段落地图→定位证据→快速判定
- 时间：开始轻掐表（先能做完，再逐步压缩）

#### 第3遍（考试型，隔7–14天）
- 目标：严格60分钟完成整套
- 复盘重点：拖时题型、NG犹豫点、填空单复数/拼写等低级错

---

### 8.5 精读的时间预算（建议）

- Passage 1：精读 20–30 分钟（或只做错题段落）
- Passage 2：精读 25–35 分钟（重点：段落匹配/多选）
- Passage 3：精读 35–60 分钟（最值钱，允许更慢）

> 如果当日精力不足：只做“错题/慢题证据句库 + 同义替换地图”，也算合格精读。
