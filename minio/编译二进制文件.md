# 编译二进制文件

## 前提条件
```shell
dnf -y install git go
```

## 命令
```shell
VERSION=2025-10-15T17:29:55Z
TAG=RELEASE.2025-10-15T17-29-55Z
COMMIT=$(git rev-parse HEAD)
YEAR=$(date +%Y)

git clone https://github.com/minio/minio.git
cd minio
git checkout "${TAG}"

env CGO_ENABLED=0 GOOS=linux GOARCH=amd64 \
  go build -trimpath \
    -ldflags "-s -w \
      -X github.com/minio/minio/cmd.Version=${VERSION} \
      -X github.com/minio/minio/cmd.ReleaseTag=${TAG} \
      -X github.com/minio/minio/cmd.CommitID=${COMMIT} \
      -X github.com/minio/minio/cmd.CopyrightYear=${YEAR}" \
      -o "minio-${TAG}" .

\cp "minio-${TAG}" ~/go/bin/minio
~/go/bin/minio -v
```
