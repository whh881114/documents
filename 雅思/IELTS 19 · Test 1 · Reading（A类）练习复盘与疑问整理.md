## 2. 通用策略：先读文章还是先看题？

结论：**大多数情况下先看题，再带目的读文章（最省脑子、也更贴合雅思机制）。**

### 推荐流程（最通用）
1. **先扫题目（不做题，只看要找什么）**
   - 圈关键词：人名/年份/地名/专有名词/比较词/极限词（only/mostly/never）
   - 识别题型：填空、段落匹配、T/F/NG、YES/NO/NG、多选等
2. **快速扫文章结构（30–90秒）**
   - 只读标题、小标题、每段首句/末句
   - 给每段写 3–7 字“标签”（段落地图）
3. **按题目去定位证据句**
   - 找到证据句就出答案，不把文章当小说精读

### 什么时候需要“先扫文章再做题”？
- **段落标题匹配/Heading matching**：需要先建立段落主旨地图
- **文章极抽象（观点推进强）**：先做 1 分钟结构扫读，避免迷路  
> 但仍不建议“慢读全文再做题”，那是用记忆硬扛检索系统，极耗时。

## 3. 通用题型：TRUE/FALSE/NOT GIVEN 与 YES/NO/NOT GIVEN 区别

### 3.1 TRUE / FALSE / NOT GIVEN（事实判断）
题干在问：**文章里的“事实信息”是否支持这个陈述？**

- **TRUE**：文章明确说了/同义改写，结论一致  
- **FALSE**：文章明确给出相反信息（证据反向打脸）  
- **NOT GIVEN**：文章没有给足信息判断真伪（不能脑补）

关键词：**fact claim（事实断言）**、客观信息、可被证据验证。

### 3.2 YES / NO / NOT GIVEN（观点态度）
题干在问：**作者/说话者是否支持这个观点？**

- **YES**：作者观点支持/赞同题干  
- **NO**：作者观点反对/不赞同题干  
- **NOT GIVEN**：作者没有表态，看不出来立场

关键词：**opinion/attitude（观点/态度）**、立场、评判、主观色彩。

### 3.3 一句话记忆法
- **T/F/NG：事实对不对**
- **Y/N/NG：作者站不站你这边**
- **NG：没证据别脑补；没提 ≠ 反对**

### 3.4 常见翻车点（必须避免）
- 把 **NOT GIVEN 当成 FALSE/NO**  
  - 反对必须有反向证据；没证据就是 NG
- 漏掉限定词：**only / mainly / most / some / in the 1990s**  
  - 限定词一漏，答案可能直接翻转
- YES/NO 题只找“事实”不找“态度信号词”  
  - 支持：argues / suggests / emphasizes / is convinced  
  - 反对：criticizes / doubts / warns against / challenges

## 4. 题型疑问与策略总结（本次遇到的重点）

### 4.1 段落信息匹配题（Which paragraph contains the following information? A–G）

**特征**：题干抽象、同义替换多，容易“看完文章记不住”。

**核心思路**：不是记全文，而是 **段落地图 + 题干信号定位证据句**。

#### 推荐流程（提速版）
1. **先给每段贴标签（30–60秒）**
   - 只读每段：首句/末句 + 扫专有名词、数字、地名
   - 用 3–7 个字写段落主题（如：现代印象/航海条件/国家利用/剿匪行动/安置政策…）
2. **把题干拆成定位信号**
   - 例如：denial / campaign / states use pirates / today view / encouraged not return / close to land
3. **先缩小范围再精读**
   - 先判断“更可能在哪两段”，再回段落里找证据句

> 结论：这题难点不是“英语读不懂”，而是“检索效率”与“同义替换敏感度”。

---

### 4.2 多选题（Choose TWO letters, A–E）

#### 题型特点
- 选项长、细节多、容易被带着全文乱找。
- 正确做法：**不是逐个验证 5 个选项**，而是快速淘汰 3 个。

#### 提速策略
1. **先把选项压缩成 2–3 个关键词标签**
2. **先在文章里锁定范围段落**
   - 题干一般明确范围（例如 Mediterranean inhabitants / ancient Greece / Iliad & Odyssey）
3. **只验证“剩下最可能的两项”**
   - 找到证据句就收工，不要企图把 5 个都证伪/证真

---

### 4.3 Summary Completion（10选6）

**题型体验**：同义替换不好找，做对 3 个。

#### 关键发现
- 很多空可以用 **语法结构 + 固定搭配（collocation）** 先锁定候选，不必死盯同义替换。
- 但最终仍需回原文快速确认，避免“多个选项都语法对”的陷阱。

#### 本次题目（31–36）推理版答案
- 31 = **G** frequent exposure（exposure to misinformation）
- 32 = **J** different ideas（had different ideas about…）
- 33 = **H** mental operation（a distinct mental operation）
- 34 = **B** additional evidence（provide evidence for…）
- 35 = **E** short period（for an extremely short period）
- 36 = **C** different locations（in different locations in the brain）

#### 考试建议（稳妥版两段式）
1. **先用语法/搭配预填**（速度快）
2. **再回原文做点位确认**（每空 10–20 秒）
   - 确认：主旨一致 + 有证据句/同义表达支撑

## 5. 练习方式与载体问题（PDF vs 纸质 / 机考）

- 使用 PDF 来回翻页定位，效率较低，容易耗时间、打断思路。
- 纸质练习的优势：
  - 圈关键词更顺手
  - 段落地图/标记更直观
- 目标：训练到 **60 分钟完成**，而不是 90 分钟慢慢做对。
