# 雅思听力 Part 1 高频场景、定位词、同义替换总表

## 使用说明

这份只保留 Part 1 内容，方便单独看、单独背。

Part 1 的核心，不是单纯“听到什么写什么”，而是：

**先判断场景 → 再预判信息类型 → 再等同义替换 → 再防改口和干扰项。**

Part 1 常见题型包括：
- 表格补全
- 表单补全
- 笔记补全
- 电话咨询 / 报名登记 / 预订信息

---

## 一、Part 1 的核心特点

### 本质
- 两个人对话
- 场景通常是日常生活场景
- 语速相对更友好
- 信息多半按顺序往下走

### 你要抓什么
- 姓名
- 地址
- 电话
- 日期
- 时间
- 价格
- 房型
- 课程
- 服务内容

### 常见难点
- 听到第一个信息就急着写，结果后面改口
- 只会等原词，不会等同义替换
- 数字、日期、拼写不够敏感
- 明明听懂大意了，但没和题目空格对上

## 二、Part 1 高频场景、定位词、同义替换

### 1. 住宿 / 酒店 / 租房 / 寄宿

#### 常见内容
- 预订房间
- 咨询住宿
- 租公寓 / 租房
- 寄宿家庭信息
- 入住时间、位置、设施、价格

#### 题目中常见词
- accommodation
- room
- apartment
- flat
- booking
- facilities
- rent
- price

#### 高频同义替换
- **book** = reserve
- **room** = accommodation / place to stay
- **flat** = apartment
- **price** = cost / fee / charge / rate
- **included** = provided / covered / comes with
- **near** = close to / within walking distance of
- **move in** = start renting / take the room
- **facilities** = amenities / services / equipment
- **available** = free / vacant / still open

#### 做题提醒
这一类最容易出现：
- 先报一个价格，后面改口
- 先说一种房型，后面换成另一种
- 先说离市中心近，后面补充其实要走十分钟

---

### 2. 旅游 / 订票 / 交通 / 行程咨询

#### 常见内容
- 订机票 / 车票
- 旅行咨询
- 报团 / 订行程
- 出发时间、到达时间、费用、票种

#### 题目中常见词
- trip
- ticket
- departure
- arrival
- tour
- journey
- travel date

#### 高频同义替换
- **trip** = journey / travel / tour
- **leave** = depart / set off / head out
- **arrive** = get in / reach
- **ticket** = fare / pass
- **return ticket** = round trip
- **book a holiday** = arrange a trip / reserve a package
- **date** = travel date / departure date
- **how long** = duration / length of stay
- **cheap** = low-cost / budget / inexpensive
- **change** = alter / switch / reschedule

#### 做题提醒
如果题目写的是 **departure**，录音里未必会说 departure，常常会说：
- leave
- set off
- go at
- head out at

---

### 3. 课程 / 活动 / 俱乐部 / 会员注册

#### 常见内容
- 报课程
- 社区活动报名
- 健身房 / 俱乐部会员登记
- 兴趣班、培训班、社团咨询

#### 题目中常见词
- course
- class
- club
- member
- register
- join
- fee
- level

#### 高频同义替换
- **course** = class / programme / training
- **join** = sign up for / enrol in / register for
- **club** = association / group
- **member** = participant / subscriber
- **begin** = start
- **finish** = end
- **weekly** = every week
- **cost** = fee / charge
- **experience** = background
- **level** = stage / standard

#### 做题提醒
这类题常见空格类型：

- 年龄
- 联系方式
- 课程时间
- 收费
- 过去经验
- 当前水平

### 4. 咨询 / 客服 / 预约 / 维修服务

#### 常见内容
- 打电话咨询服务
- 预约时间
- 报修
- 查询某项服务是否可用
- 登记个人信息

#### 题目中常见词
- service
- repair
- problem
- appointment
- enquiry
- address
- details

#### 高频同义替换
- **ask about** = enquire about / find out about
- **problem** = issue / trouble / fault
- **repair** = fix
- **appointment** = booking / reservation / scheduled time
- **service** = help / assistance
- **address** = location / place
- **contact number** = phone number / telephone number
- **details** = information / particulars
- **free** = available / vacant
- **confirm** = check / make sure

#### 做题提醒
这类很爱考：
- 电话号码
- 门牌号
- 预约时间
- 维修日期
- 上门地址
- 费用

---

### 5. 求职 / 志愿者 / 申请表

#### 常见内容
- 兼职申请
- 志愿者报名
- 工作咨询
- 登记可上班时间、经验、技能

#### 题目中常见词
- application
- job
- position
- work experience
- skills
- available days
- salary

#### 高频同义替换
- **job** = position / role / post
- **apply for** = submit an application for
- **work experience** = background / previous experience
- **skills** = abilities / qualifications
- **part-time** = casual / temporary
- **available** = free / able to work
- **salary** = pay / wages
- **manager** = supervisor
- **CV** = resume
- **reference** = referee / recommendation

#### 做题提醒
最常见陷阱：
- 先说 Friday，后面改成 Saturday
- 先说 full-time，后面纠正为 part-time
- 先给一个时薪，后面又补充试用期薪资

---

### 6. 购物 / 订单 / 运送 / 报价

#### 常见内容
- 订购商品
- 配送
- 询价
- 报价单 / 订单表
- 数量、单价、总价

#### 题目中常见词
- order
- quote
- delivery
- shipping
- item
- quantity
- payment

#### 高频同义替换
- **order** = purchase / booking
- **price** = quote / quotation / charge
- **send** = deliver / ship
- **buy** = order / get
- **item** = product / goods
- **amount** = quantity / number
- **pay** = make payment
- **deposit** = advance payment
- **extra** = additional
- **total** = overall cost / final amount

#### 做题提醒
要特别区分：
- 单价 vs 总价
- 押金 vs 全款
- 送货日期 vs 下单日期
- 数量 vs 包装数

## 三、Part 1 最常考的信息类型同义替换

### 1. 人名 / 联系方式
- **surname** = family name / last name
- **first name** = given name
- **full name** = complete name
- **contact number** = phone number / telephone number
- **email** = email address

### 2. 地址 / 地点
- **address** = home address / postal address
- **postcode** = postal code / zip code
- **near** = close to / next to
- **opposite** = across from
- **behind** = at the back of
- **in the city centre** = central / downtown
- **on the corner** = at the corner of

### 3. 时间 / 日期
- **begin** = start
- **finish** = end
- **at 7** = at seven / seven o'clock
- **on Friday** = this Friday / Friday morning / Friday afternoon
- **for 3 days** = lasting three days / a three-day period

### 4. 钱 / 费用
- **price** = cost / charge / fee / rate
- **cheap** = inexpensive / budget / low-cost
- **pay** = make payment
- **deposit** = advance payment
- **free** = no charge

### 5. 喜好 / 需求
- **want** = would like / be looking for
- **need** = require
- **prefer** = would rather have
- **suitable** = appropriate / right
- **include** = contain / involve / cover

## 四、Part 1 做题时的预判思路

### 看到题目时，不要直接等原词

你要在脑子里自动展开替换。

#### 例子 1
题目写：

**accommodation**

你耳朵里要准备等这些表达：
- room
- flat
- apartment
- place to stay

#### 例子 2
题目写：

**cost**

你耳朵里要准备等这些表达：
- price
- fee
- charge
- rate

#### 例子 3
题目写：

**starts**

你耳朵里要准备等这些表达：
- begins
- opens
- starts at

#### 例子 4
题目写：

**close to the station**

你耳朵里要准备等这些表达：
- near the station
- next to the station
- within walking distance of the station

#### 例子 5
题目写：

**part-time job**

你耳朵里要准备等这些表达：
- weekend work
- temporary work
- casual work

## 五、Part 1 复盘模板

你以后每做完一套 Part 1，可以按这个格式整理：

### 1. 场景
例如：
- hotel booking
- job application
- club membership
- travel enquiry

### 2. 关键信息类型
例如：
- name
- address
- date
- cost
- room type
- opening time

### 3. 同义替换
例如：
- price = cost / fee / charge
- room = accommodation / place to stay
- start = begin
- close to = near

### 4. 陷阱点
例如：
- 信息改口
- 数字听错
- 单复数没听清
- 拼写漏字母
- 题目和录音不是原词对应

## 六、最后的核心提醒

Part 1 最容易出问题的，不是词汇量本身，而是这几个地方：

1. **没先判断场景**
2. **只会等原词，不会等替换**
3. **听到第一个信息就写，没等后面改口**
4. **对数字、日期、拼写不够敏感**
5. **听懂了大意，但没和空格真正对上**

所以做 Part 1 时，脑子里最好始终跑这条线：

**先看题目判断场景，再预判信息类型，再等同义替换，最后再确认有没有改口和干扰项。**

## 七、建议背诵顺序

建议你先背这 6 组：

1. 住宿
2. 旅游交通
3. 课程活动
4. 预约服务
5. 求职申请
6. 购物订单

把这 6 类背熟，Part 1 绝大多数日常场景你都不会太陌生。
