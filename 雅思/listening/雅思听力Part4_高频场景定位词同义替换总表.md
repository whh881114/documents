# 雅思听力 Part 4 高频场景、定位词、同义替换总表

## 使用说明

这份只保留 Part 4 内容，方便单独看、单独背。

Part 4 的核心是：

**结构、分类、因果、过程、研究结论。**

做题时脑子里最好跑这条线：

**先看标题和小标题 → 再跟住结构词 → 再抓定位词和替换。**

# 七、Part 4 高频场景、定位词、同义替换

## 1. 动物 / 环境 / 生态类讲座

### 常见内容
- 某种动物行为
- 栖息地
- 生存方式
- 环境变化影响
- 保护措施

### 高频定位词
- habitat
- species
- population
- migration
- breeding
- survival
- climate
- environment
- food source
- predator
- conservation
- adaptation

### 高频同义替换
- **habitat** = natural environment / living area
- **species** = type / kind
- **population** = number of animals
- **migration** = seasonal movement
- **breeding** = reproduction
- **survival** = staying alive
- **food source** = source of food
- **predator** = animal that hunts others
- **conservation** = protection
- **adaptation** = adjustment to conditions

### 做题提醒
这类最爱考：
- 特征分类
- 原因
- 影响
- 适应方式
- 研究发现

---

## 2. 历史 / 考古 / 社会发展类讲座

### 常见内容
- 某地历史发展
- 古代生活方式
- 技术演变
- 社会变化
- 某项发现的意义

### 高频定位词
- evidence
- records
- excavation
- ancient
- development
- trade
- tools
- settlement
- social structure
- changes over time
- influence

### 高频同义替换
- **evidence** = proof / findings
- **records** = written documents
- **excavation** = digging at a site
- **ancient** = very old / from early times
- **development** = growth / progress
- **trade** = exchange of goods
- **settlement** = place where people lived
- **influence** = effect / impact

### 做题提醒
别把所有年份都当答案。  
Part 4 很多时候年份只是背景，真正考的是：
- 原因
- 变化
- 结果
- 学者如何解释

- planning
- efficiency
- market

- cost

### 高频同义替换
- **production** = making / manufacture
- **process** = procedure / stages
- **distribution** = delivery / supply
- **design** = layout / plan
- **planning** = organisation / preparation
- **efficiency** = how well something works
- **market** = demand area / buyers

### 做题提醒
这类题你要特别敏感：
- 阶段顺序

## 4. 心理学 / 教育 / 研究方法类讲座

### 常见内容
- 学习方式
- 儿童发展
- 实验设计
- 研究对象
- 结果分析

### 高频定位词
- experiment
- participants
- method
- result
- findings
- behaviour
- memory
- learning
- observation
- compare
- conclusion
- effect

### 高频同义替换
- **experiment** = test / study
- **participants** = subjects / people involved
- **method** = approach / procedure
- **result** = outcome
- **findings** = results discovered
- **behaviour** = actions
- **memory** = ability to remember
- **observation** = watching carefully
- **compare** = look at differences and similarities
- **conclusion** = final finding
- **effect** = impact / influence

### 做题提醒
这类题最容易听着像“都懂”，结果一个空都填不出来。  
因为它真正考的是：
- 研究目的
- 方法步骤
- 结果含义
- 哪个变量有影响

# 十二、三部分最常见的陷阱

## 1. 改口
- 先说一个方案，后面换掉
- 先说一个地点，后面补充真正位置
- 先说一个原因，后面说“其实更重要的是……”

### 提醒
答案常在后面，不在前面。

---

## 2. 干扰项提前出现
- 选项 A 的词先出来，不代表就是答案
- 录音可能先提到错误选项，再解释为什么不用它

### 提醒
尤其是选择题，别听到关键词就抢答。

---

## 3. 只盯单词，不跟逻辑
- 你单词听懂了
- 但你没听懂“谁反对谁”“哪个是最终决定”“因果关系是什么”

### 提醒
Part 3 和 Part 4 更看逻辑，不只是词汇。

---

## 4. 单复数和词性错误
- 题目要名词，你填了动词
- 题目暗示复数，你只写单数
- 题目空的是形容词，你填了名词

### 提醒
做空格题时，必须先看语法位置。

---

## 5. 标题和小标题没看
- 你没看清是“advantages”还是“problems”
- 你没看清这一栏是“location”还是“purpose”

### 提醒
标题就是范围，不看等于白听。

---

# 十六、最后的核心提醒

你做 Part 2 / 3 / 4，不要再用 Part 1 那种“等关键词蹦出来”的思路硬扛。

后面三部分越来越看这些能力：

1. **能不能快速判断场景**
2. **能不能圈出真正有用的定位词**
3. **能不能听懂替换，而不是只等原词**
4. **能不能跟住转折、因果、对比、分类**
5. **能不能在干扰项出来时忍住，不要抢选**
